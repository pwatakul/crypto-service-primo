
## Description

This is crypto-service project for interview.

## Installation

```bash
$ npm install
```

## Update .env
- Copy the .env file
```bash
$ cp .env.example .env
```

- Get the public and private key https://cryptotools.net/rsagen
- Update its in the .env file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm test

```
